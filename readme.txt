== Installation ==
1. Téléchargez les fichiers du plugin depuis inkreez.com, ou installez le plugin directement via l'écran des plugins WordPress.
2. Activez le plugin via l'écran 'Plugins' dans WordPress.
3. Utilisez l'écran des paramètres Inkreez pour configurer le plugin.
